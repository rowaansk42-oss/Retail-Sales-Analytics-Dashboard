# Retail Sales MIS Dashboard

## Overview

The Retail Sales MIS Dashboard is an interactive Power BI dashboard designed to analyze retail sales performance using Excel data. It provides business insights through KPIs, charts, and interactive filters, helping users monitor sales trends and customer behavior.

## Dashboard Preview

![Dashboard](Dashboard.png)

---

## Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

---

## Key Features

- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value
- Average Quantity Sold
- Monthly Sales Trend Analysis
- Product Category-wise Sales
- Gender-wise Sales Analysis
- Age Group-wise Sales Analysis
- Interactive Slicers (Month & Product Category)

---

## Business Insights

- Identified the highest-performing product category.
- Analyzed customer purchasing patterns across different age groups.
- Compared sales distribution by gender.
- Monitored monthly sales trends using interactive visualizations.
