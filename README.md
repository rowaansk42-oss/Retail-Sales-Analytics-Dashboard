# Retail-Sales-Analytics-Dashboard
Interactive MIS Dashboard built using Power BI and Excel for retail sales performance analysis.

## Project Overview

This project is an interactive Retail Sales MIS Dashboard developed using Power BI and Excel.

The dashboard helps analyze sales performance, customer behavior, and product-wise business insights through interactive visualizations.

---

## Tools Used

- Power BI
- Microsoft Excel
- DAX
- Data Visualization

---

## Features

- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value
- Average Quantity Sold
- Monthly Sales Trend
- Sales by Product Category
- Sales by Gender
- Sales by Age Group
- Interactive Filters
